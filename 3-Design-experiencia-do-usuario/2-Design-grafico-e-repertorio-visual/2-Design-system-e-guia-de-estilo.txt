    Design systems e guias de estilos

    Design systems
        É um conjunto pré construído de elementos, padrões e regras para a criação de interfaces
    
    Por que?
        Maior agilidade no processo de prototipagem
        Maior consistência
        Não ter que começar do zero cada novo elemento

    O que deve ter?
        Fontes utilizadas, hierarquias, funções, paleta cromática, características básicas de elementos e regras básicas de ícones, imagens, grid e layout
        Complete o que for necessário
    
    Exemplos
        Material design (Google)
        Bold (Ufsc)
        Adele (banco de design systems open source)

    Tipografia
        São os estilos das letras

        Conceitos básicos
            Hierarquia de fontes: defina a hierarquia entre os diferentes elementos e quais configurações de fonte utilizam. (fonte, tamanho, estilo, etc)

            Tamanho de fonte
                Será que diferentes dispositivos conseguem enxergar a fonte daquele texto
                Fonte tão grande que as palavras não cabem no grid
                Texte no contexto relacionados
            
            Legibilidade
                Textos com serifa são melhor
                Melhor para dispositivos móveis
            
            Leiturabilidade
                Textos com serifa são melhor
                Quão fácil é a leitura em textos longos

            Tipos de fontes
                Famílias tipográficas
                Manuscritas
                (Com serifa:
                    Orientar melhor o olho para identificar esses caracteres)
        
        Recomendações para projetos
            Utilize apenas uma ou duas familias tipográficas por projeto
            Variedade de pesos são mais versateis** (bold, italic, etc)
            Fontes que tenha suporte para acentos e adaptáveis para a língua portuguesa
            Não use fontes clichês (comic sans ou papirus, por exemplo)
            Use fontes substitutas: patrick hand e antic didone
            Exemplos para uso: helvetica neue (fonte clássica) e roboto / Garamond e Lobster

    Reengenharia de processos de negócio
        Um processo de negócio é um conjunto de tarefas relacionadas de maneira lógica, desempenhadas para obeter um resultado de negócio definido.
        A reengenharia é a busca e a implementação de mudanças radicais nos processos de negócio para obter resultados excelentes 
    
    Reengenharia de software  
        É o trabalho de reforma de um software
        Para melhorar o desempenho, a manutenibilidade, a arquitetura, a seguração e promover a transformação digital

    Processo de reengenharia de software
        Análise de inventário
            Lista das aplicações da org
            Forma com elas estão integradas
        Reestruturação de documentos
            Falta de documentação é um problema recorrente
        Engenharia reversa
            Processo de recuperação do projeto
        Reestruturação do código
            Refatoração
            Melhoria da qualidade do código
            Podemos usar o SonarQube (por exemplo)
        Reestruturação dos dados
            Mudanças de arquitetura e também no código
        Engenharia direta
            Recuperação do projeto do software com a sua respectiva alteração
